1. Markdown

第一课当然是比较简单的，让大家了解一下 Markdown 这种语法。这是一种非常方便而且应用广泛的一种输入方式
希望大家能够将该语法作为自己输入和记录的常用方法，自己做记录的时候尽量不要使用 Word

2. 工具

PC 端各个系统强烈推荐 Visual Studio Code（简称 VS Code），点击 download 即可下载
新建后缀为.md文件之后即可开始编写，右上角选择打开侧边预览即可看到效果
